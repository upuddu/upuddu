### About

I build intelligent systems and the interfaces that make them feel alive — transformer
models and numerical libraries written in low-level languages, embedded hardware, and
interactive simulations. My work spans machine-learning research, systems programming,
and the occasional soldering iron.

| Project | What it is |
| --- | --- |
